# RestaurantPro Demo

## Déploiement Rapide

1. Uploadez ce dossier sur GitHub
2. Connectez à Vercel
3. Déployez avec les paramètres par défaut

## Comptes de Test
- Chaîne: chain@demo.com / demo123
- Gastro: gastro@demo.com / demo123

## Configuration Vercel
- Build Command: `npm run build`
- Output Directory: `frontend/build`
- Variables d'environnement:
  - NODE_ENV=demo
  - DEMO_MODE=true
